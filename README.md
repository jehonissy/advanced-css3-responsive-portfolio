# Advanced CSS3 Responsive Portfolio

A fully responsive portfolio website built using HTML5, CSS3, CSS Grid, Flexbox, and JavaScript with dark/light theme support.

## Features

- CSS Grid layouts
- Flexbox components
- Mobile-first responsive design
- Dark/Light theme
- Accessibility-friendly structure

## Technologies Used

- HTML5
- CSS3
- JavaScript
- CSS Grid
- Flexbox
